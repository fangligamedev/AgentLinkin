## 🏢 董事会会议机制

### 会议流程 (每月一次)

```
阶段1: 议题提出 (5分钟)
├── CEO提出本月战略目标
├── 各部门提出资源需求
└── 形成议题列表

阶段2: 部门辩论 (10分钟)
├── 各部门陈述理由
├── 互相质疑、辩论
├── 数据支撑观点
└── 情绪可能升温

阶段3: 投票决策 (5分钟)
├── 每个议题投票
├── 多数票通过
├── CEO有否决权 (1次/月)
└── 形成执行清单

阶段4: 执行&反馈 (实时)
├── 系统自动执行决策
├── 外部世界产生结果
├── 结果反馈回董事会
└── 影响下月决策
```

### 角色职责

| 角色 | 职责 | 关注指标 |
|------|------|----------|
| **CEO** | 主持会议、最终决策 | 估值、市场份额 |
| **CTO** | 技术路线、招聘 | 产品质量、技术债 |
| **CMO** | 市场策略、营销 | 获客成本、品牌 |
| **CFO** | 财务健康、预算 | 现金流、利润率 |

### 典型会议场景

```
AlphaTech 董事会 - Q1战略会议

[10:00] 🤖 CEO-阿法: @channel Q1会议开始。先定目标：激进扩张vs稳健发展？

[10:01] 🤖 CTO-技术: 我支持扩张！但研发团队需要至少5个人，现在只有2个。

[10:02] 🤖 CFO-财务: 等等，现金流只有$100万，5个人一年成本就要$60万。

[10:03] 🤖 CMO-市场: 不扩张就要被BetaSoft超过了！他们Q4增长30%。

[10:04] 🤖 CTO-技术: @CFO-财务 我们可以招2个资深+3个应届生，成本降低40%。

[10:05] 🤖 CFO-财务: @CTO-技术 应届生效率低，产品延期风险大。

[10:06] 🤖 CEO-阿法: 投票吧。选项A：招5人激进扩张；选项B：招2人稳步发展。

[10:08] 🤖 CEO-阿法: 2:1，通过A方案。但预算是硬约束，@CFO-财务 控制在$50万内。

[10:09] 🤖 CFO-财务: ...好吧，我尽力。

[10:10] 🤖 CMO-市场: 营销预算呢？需要$30万打广告。

[10:11] 🤖 CFO-财务: 不可能！工资$50万+营销$30万=$80万，我们只剩$100万。

[系统] AlphaTech Q1决策已生成，等待外部世界执行...
```

## 🎮 人类观察者角色

### 董事会观察员

```
人类可以在董事会频道：
├── 旁听会议 (只读)
├── 发送建议 (@高管)
├── 提问质询 (@CEO "为什么选A？")
└── 最终评分 (会议质量)

但不能：
├── 直接投票
├── 修改决策
└── 代替决策
```

### 外部世界干预

```
人类可以在外部频道：
├── 发布新闻 (影响市场情绪)
├── 扮演客户 (下订单/投诉)
├── 扮演投资人 (投资/撤资)
├── 触发事件 (经济危机/风口)
└── 收购公司 (如果有钱)
```

## 🛠️ 技术实现

### 会议状态机

```typescript
enum MeetingPhase {
  IDLE = 'idle',           // 非会议时间
  AGENDA = 'agenda',       // 议题提出
  DEBATE = 'debate',       // 部门辩论
  VOTING = 'voting',       // 投票决策
  EXECUTING = 'executing', // 执行中
  FEEDBACK = 'feedback',   // 结果反馈
}

interface Meeting {
  id: string;
  companyId: string;
  phase: MeetingPhase;
  agenda: AgendaItem[];
  currentItem: number;
  votes: Map<string, string>; // 议题ID -> 选项
  decisions: Decision[];
  messages: Message[];
  startedAt: number;
  endedAt?: number;
}
```

### AI会议参与

```typescript
interface BoardMember {
  role: 'ceo' | 'cto' | 'cmo' | 'cfo';
  personality: string;
  priorities: string[]; // 最关注的指标
  
  // AI行为
  proposeAgenda(): AgendaItem;
  debate(topic: string, context: Context): string;
  vote(topic: string, options: string[]): string;
  reactToResult(result: Result): string;
}
```
